# httprs

> **Status:** Pre-alpha. Not ready for production use.

An experimental HTTP library for Python built with Rust.

## Experimenting with httprs

### Install

```bash
pip install --pre httprs
```

### Example

```python
import httprs

httprs.get("https://postman-echo.com/get").json()
```

## Goals

- Simple
- Fast
- Open
